## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
https://github.com/Mariogil2001/blog-svelte.git
```

### 📦️ Install dependencies

```sh
pnpm i
```

### 💿️ Run the development server

```sh
pnpm run dev
```
